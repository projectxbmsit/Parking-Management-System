# MAD-Mini-Project

### Note for pushing
To change the branch of your local repo:
`$ git checkout branchName`

To add files to the staging area
`$ git add fileName`

To commit
`$ git commit -m "commit msg"`

To push to your branch
`$ git push origin branchName`
